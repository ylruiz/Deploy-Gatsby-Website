What Web Design and Web Development mean?

Web Development

Web development involves building, creating, and maintaining websites. Among web professionals, this term usually refers to the main non-design aspects of building websites. It includes several tasks, such as web engineering, web design, web content development, client-side/server-side scripting, web server, and network security configuration.

Web Design

Web design is an activity that consists in the creation of websites. It includes different areas, like web graphic design, interface design, and user experience design. It covers other aspects such as navigability, interactivity, usability, and information architecture. Moreover, web design involves media interaction, for example, audio, text, image, and search engine optimization.

What involves to be a web designer and a web developer?

There are two primary jobs involved in creating a website: web designers and web developers, who often work closely together on a website. The following table shows the main differences between each other.

AspectWeb DesignerWeb Developer
DefinitionWeb designers are responsible for the visual aspect of the website, which includes the layout, coloring, and typography. It is also great if they have a working knowledge of markup languages, such as HTML and CSS.Web developers turn the design -the website structure, design markups, and prototypes- into code. As a result, they transform the vision into a fully functional website.
SkillsWeb designers are responsible for the interface of a website, which means that they should consider developing the following set of skills:
- Visual Design: The most common tools in the visual design are grid systems, typography, color theory, and web fonts.
- User experience (UX): A web designer who understands UX practices always will create a website from a user-first perspective.
- Communication, Time-Management, Proactivity, Creativity, and Digital Marketing skills
- HTML and CSS: Those are expected coding languages that a web designer should know.		Here are some of the most popular skills to have in this role:
- HTML/CSS/JavaScript: Those are programming languages that every web developer needs to master.
- CSS and JavaScript Frameworks: Bootstrap, Foundation, Node.js, React.js, Vue.js, Angular.js
- CSS preprocessing: SASS, LESS
- Back-end basics: APIs, databases, and cybersecurity.
- Testing and debugging: It is essential to know how to test your code for any possible errors.
ToolsSeveral types of tools make web designers' labor more easer, for example:
- Image Editor: Adobe Photoshop, Adobe Illustrator
- Wireframe Tool: Sketch, Balsamiq
- Stock photography resources: Unsplash, iStock
- Image optimization: ResizeImage.net, TinyPNG
- Typographies and icons libraries: Font Awesome, Google Fonts
- Prototyping software: In Vision Studio, Adobe XD		Some tools that web developers use are the next:
- Integrated Development Environment (IDE): Visual Code Studio
- Version Control Platform: GitHub, GitLab
- Browser developer tools: Chrome DevTools, Firefox DevTools
- API & Testing Tools: Postman, LambdaTest
- Collaboration Tools: JIRA, Zoom

Due to web designers and web developers frequently have close collaboration, it is important to note that they utilize a certain set of tools that facilities communication between each other. For instance:

  • Design-Developer collaboration tool: Figma, In Vision
  • Project management software: Teamwork, Trello
  • Team chat software: Skype, Slack

Where can you learn web design and web development?

There are different ways to learn about web design or web development. For instance:

For designers

  • The Academy of Art University
  • Subscribe to platforms like Dribbble and Behance.
  • Go through all of the tutorials provided by your preferred design software.
  • Subscribe to blogs and YouTube channels that specialize in web design tips and tutorials, such as Smashing Magazine and WebDesignerDepot.

For Developers

  • Coursera
  • FreeCodeCamp
  • EggHead
  • LearnCode.academy on YouTube

I also recommend you to follow the tutorials of the tools or frameworks that you want to learn. Additionally, enrolling in meetups is a great way to maintain yourself learning new stuff.
